API Dokumentasjon

Her får du en oversikt over alle endepunkter og datamodeller i SportsWorld API

Finance

Det er 1 oversikt over økonomien, da det er til den ene bedriften

Datatype-oversikt for økonomi

Key Type Beskrivelse
id int Id for en bedrift
moneyLeft double Beløpet som er igjen (etter feks kjøp av spiller)
NumberOfPurchases int Antall kjøp (av spillere)
moneySpent double Beløpet som er brukt (ved kjøp av spiller)

GET /api/finance

Henter en fullstendig oversikt over økonomi

GET http://localhost:5236/api/finance

JSON-resons:

            {
                "id": 1,
                "moneyLeft": 10000,
                "NumberOfPurchases": 3,
                "moneySpent": 50000"
            }

PUT /api/finance/{id}

Oppdaterer et finansobjekt (feks ved justering av penger etter kjøp)

PUT http://localhost:5236/api/finance/1

Eksempel på JSON-resons etter lån av penger og kjøp av spiller 

            {
                "id": 1,
                "moneyLeft": 555000005,
                "numberOfPurchases": 5,
                "moneySpent": 445000000,
                "amount": 0,
                "price": 0
            }

Athletes

Det er 8 spillere i API-et

Datatype-oversikt for spillere

Key Type Beskrivelse
id int Iden til spilleren
name string Navnet på spilleren
gender string Kjønn på spilleren
price double Pris på spilleren
image string Bilde til spilleren
purchaseStatus bool Om spilleren er kjøpt eller ikke (default: false, altså ikke kjøpt)

Get en enkel utøver - GET/api/athlete/{id}

Du kan hente en enkelt spiller ved å legge til id som et parameter: /api/athlete/1

GET http://localhost:5236/api/athlete/1

JSON-resons:

            {
                "id": 1,
                "name": "Lionel Messi",
                "gender": "Male",
                "price": 500000,
                "image": "messi.jpg",
                "purchaseStatus": true
            }

Andre CRUD operasjoner

Post legge til en ny spiller - POST /api/athlete

Ekesmpel på JSON-resons for en ny spiller: 

              {
                "id": 9,
                "name": "Pedri",
                "gender": "Mann",
                "price": 85000000,
                "image": "pedri.jpg",
                "purchaseStatus": false
            }

Put oppdarere en spiller etter id - PUT /api/athlete/{id}

Delete slette en spiller etter id - DELETE /athelte/{id}
