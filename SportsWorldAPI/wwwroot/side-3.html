Om siden

SportsWorld API er laget basert på fotballspillere fra flere klubber og med en økonomioversikt. Du får tilgang til 8 spillere: id, navn, kjønn, bilde, pris, og om spilleren er kjøpt eller ikke. I økonomioversikten er man antall spillere som er kjøpt, hvor mye penger du har brukt, og hvor mye penger du har igjen.

Om oss

Vi er to studenter som studerer informasjonsteknologi, og dette er vår besvarelse på eksamen i emnet webutvikling.

Teknisk

Vi har brukt HTML5, CSS3, CSSFrameWork: Tailwind CSS, React, JavaScript (ES6+), .NET/# Web API med SQLLite Database for hele prosjektet (backend og frontend).
